Another database manipulation project on SQL database
through php and ajax

1.  A simple bootstrap framework has been used design the      layout.

2.  The web app displays the content of the data base 
    and the buttons perform create read update or delete 
    operations

3.  The speciality of ajax is that it updates the data 
    without page reloading it enhances user
    experience

4.  In the plain php crud operation we have seen the page 
    reloads whenever data is updated ...
